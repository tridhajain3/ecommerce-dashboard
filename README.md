# 📊 E-Commerce Analytics Dashboard

An interactive web-based analytics dashboard built using **Python**, **Streamlit**, **Pandas**, and **Plotly** for analyzing e-commerce sales data. The dashboard provides business insights through interactive filters, KPI cards, visualizations, and downloadable reports.

## 🚀 Live Demo

https://ecommerce-dashboard-7h5eeapeawvumd96tjrvsc.streamlit.app/

---

## 📌 Project Overview

This project transforms raw e-commerce transaction data into an interactive business intelligence dashboard that enables users to:

* Monitor sales performance
* Analyze customer purchasing patterns
* Identify top-performing products
* Compare category-wise revenue
* Explore trends over time
* Export filtered reports

The dashboard is designed with a clean and intuitive interface focused on readability and user experience.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/tridhajain3/ecommerce-dashboard.git
```

Navigate to the project:

```bash
cd ecommerce-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
